## Atomic note principle

An atomic note contains exactly one idea, written so it can be
understood without reading any other note. It is indivisible —
splitting it further would destroy its meaning.

The test for atomicity: cover the title and read only the body.
If you need to look something up to understand it, the note is
not atomic enough. Rewrite it with more self-contained context.

Atomic notes produce sharp embedding vectors because each chunk
maps to a single concept in semantic space. Mixed-topic notes
produce diluted vectors that retrieve poorly.
